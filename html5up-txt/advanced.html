Full OLL and PLL

You might want to take some time off for this one

  • 5 days ago
  • 1,024
Hover over the notation of some algorithms to get a helpful note!

At this point, you will have already learned the beginner CFOP method, which includes 2-Look OLL and PLL using 10 and 6 algorithms, respectively. Now, you will learn full OLL using 57 algorithms, and full PLL using 21 algorithms. It is recommended to learn both at the same time, so that you can learn the common OLLs and PLLs first. The 16 algorithms you already know are still good. Now you just need to learn 62 more :)
